  Welcome to Kodi Home Theater Software!
</h1>
<p>Kodi is an award-winning <strong>free and open source</strong> software media player and entertainment hub for digital media. Available as a native application for <strong>Android, Linux, BSD, macOS, iOS, tvOS and Windows operating systems</strong>, Kodi runs on most common processor architectures.</p>
<p>Created in 2003 by a group of like minded programmers, Kodi is a non-profit project run by the XBMC Foundation and developed by volunteers located around the world. More than 500 software developers have contributed to Kodi to date, and 100-plus translators have worked to expand its reach, making it available in more than 70 languages.</p>
<p>While Kodi functions very well as a standard media player application for your computer, it has been designed to be the perfect companion for your HTPC. With its <strong>beautiful interface and powerful skinning engine</strong>, Kodi feels very natural to use from the couch with a remote control and is the ideal solution for your home theater.</p>
<h2><a id="user-content-give-your-media-the-love-it-deserves" class="anchor" aria-hidden="true" href="#give-your-media-the-love-it-deserves"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Give your media the love it deserves</h2>
<p>Kodi can be used to play almost all popular audio and video formats around. It was designed for network playback, so you can stream your multimedia from anywhere in the house or directly from the internet using practically any protocol available.</p>
<p>Point Kodi to your media and watch it <strong>scan and automagically create a personalized library</strong> complete with box covers, descriptions, and fanart. There are playlist and slideshow functions, a weather forecast feature and many audio visualizations. Once installed, your computer or HTPC will become a fully functional multimedia jukebox.</p>

<h2><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Getting Started</h2>
<p>Kodi's developers work hard to make it support a large range of devices and operating systems. We provide final as well as development builds. To get started, head over to the <strong><a href="https://kodi.tv/download" rel="nofollow">downloads section</a></strong> and simply select the platform that you want to install it on. A <strong><a href="https://kodi.wiki/view/quick_start_guide" rel="nofollow">quick start guide</a></strong> to help you get acquainted with Kodi is available in our wiki.</p>
<h2><a id="user-content-how-to-contribute" class="anchor" aria-hidden="true" href="#how-to-contribute"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>How to Contribute</h2>
<p>Kodi is created by users for users and <strong>we welcome every contribution</strong>. There are no highly paid developers or poorly paid support personnel on the phones ready to take your call. There are only users who have seen a problem and done their best to fix it. This means Kodi will always need the contributions of users like you. How can you get involved?</p>
<ul>
<li><strong>Coding:</strong> Developers can help Kodi by <strong><a href="https://github.com/xbmc/xbmc/issues">fixing a bug</a></strong>, adding new features, making our technology smaller and faster and making development easier for others. Kodi's codebase consists mainly of C++ with small parts written in a variety of coding languages. Our add-ons mainly consist of python and XML. For more information, please have a look at our <strong><a href="/xbmc/xbmc/blob/master/docs/CONTRIBUTING.md">contributing guide</a></strong>.</li>
<li><strong>Helping users:</strong> Our support process relies on enthusiastic contributors like you to help others get the most out of Kodi. The #1 priority is always answering questions in our <strong><a href="https://forum.kodi.tv/" rel="nofollow">support forums</a></strong>. Everyday new people discover Kodi, and everyday they are virtually guaranteed to have questions.</li>
<li><strong>Localization:</strong> Translate <strong><a href="https://www.transifex.com/teamxbmc/kodi-main/" rel="nofollow">Kodi</a></strong>, <strong><a href="https://www.transifex.com/teamxbmc/xbmc-addons/" rel="nofollow">add-ons</a></strong> and <strong><a href="https://www.transifex.com/teamxbmc/xbmc-skins/" rel="nofollow">skins</a></strong> into your native language.</li>
<li><strong>Add-ons:</strong> <strong><a href="https://kodi.tv/addons" rel="nofollow">Add-ons</a></strong> are what make Kodi the most extensible and customizable entertainment hub available. <strong><a href="https://kodi.tv/create-an-addon" rel="nofollow">Get started building an add-on</a></strong>.</li>
<li><strong>Documentation:</strong> Kodi's <strong><a href="https://kodi.wiki/" rel="nofollow">wiki pages</a></strong> are the hub for information about Kodi and surrounding ecosystem. Help make our documentation better by writing new content or correcting existing material.</li>
</ul>
<p><strong>Not enough free time?</strong> No problem! There are other ways to help Kodi.</p>
<ul>
<li><strong>Spread the word:</strong> Share Kodi with the world! Tell your friends and family about how Kodi creates an amazing entertainment experience. Stay up to date on the latest stories about Kodi reading our <strong><a href="https://kodi.tv/blog" rel="nofollow">news</a></strong> section, follow us on <strong><a href="https://twitter.com/koditv" rel="nofollow">Twitter</a></strong> and <strong><a href="https://www.facebook.com/XBMC/" rel="nofollow">Facebook</a></strong>, or <strong>star Kodi's repo</strong> if you want to follow development.</li>
<li><strong>Donate:</strong> We are always happy to receive a <strong><a href="https://kodi.tv/contribute/donate" rel="nofollow">donation</a></strong>. Donations are typically used for travel to attend conferences, any necessary paperwork and legal fees, and the yearly XBMC Foundation Developers Conference, where a great deal of coding and planning for the following year occurs. Donations may also be used to purchase necessary hardware and licenses for developers, along with t-shirts, stickers, and other accessories for conferences.</li>
<li><strong>Buy Kodi merchandise:</strong> Purchasing Kodi gear helps just as much as a donation, and you get something in return! Checkout our <strong><a href="https://kodi.tv/store" rel="nofollow">store</a></strong> for Kodi branded gear. We regularly add new products so check back often.</li>
</ul>
